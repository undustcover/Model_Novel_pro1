## 2. 深度人物关系链 (Deep Relationship Chains)

> 格式: **[角色A] -> [角色B]**

### 核心双人组 (The Duo)

**[林野 (Lin Ye)] -> [苏浅 (Su Qian)]**
*   **表面关系**: 被“那个奇怪转校生”纠缠的倒霉蛋。
*   **核心利益共同点**: 活下去（虽然林野目前只是想考大学，苏浅想救世界）。
*   **深层矛盾/隐雷**: **混沌 vs 秩序**。林野的每一次“直觉行动”都在破坏苏浅精心计算的“最优解”，但又往往误打误撞解决了问题。苏浅觉得他是疯子，他觉得苏浅是控制狂。
*   **当前状态**: **[没头脑] 状态**。林野完全没意识到苏浅的大佬身份，只觉得她是个脑子有点问题的富家千金。

**[苏浅 (Su Qian)] -> [林野 (Lin Ye)]**
*   **表面关系**: 恨铁不成钢的“保姆” / 暴躁的“攻略组组长”。
*   **核心利益共同点**: 必须依靠林野的“变量”属性来打破第 100 次轮回的死局。
*   **深层矛盾/隐雷**: **牺牲 vs 存活**。在苏浅的攻略里，为了胜利可能需要献祭掉林野在意的人，这是林野绝对无法接受的底线。
*   **当前状态**: **[不高兴] 状态**。每天都在血压飙升中度过，看着林野在作死的边缘疯狂试探。

### 主角团内部羁绊 (The Team Dynamics)

**[顾子轩] -> [林野]**
*   **关系**: **投资人 -> 挚友**。最初只是利用林野的技术对抗泰坦，后来被林野的人格魅力（或者说是傻劲）折服。
*   **互动模式**: 顾子轩负责砸钱平事，林野负责惹事。顾子轩经常吐槽林野“这种穷酸的维修方式”，但每次关键时刻都靠林野的维修翻盘。

**[艾薇] -> [林野]**
*   **关系**: **病患 -> 依赖者**。林野是唯一能缓解她脑芯片过载痛苦的人（通过物理维修手段误打误撞）。
*   **互动模式**: 艾薇嘴上毒舌“笨蛋林野”，实际上是林野的头号迷妹。谁敢欺负林野，她就黑掉谁的银行账户。

**[陆霆] -> [顾子轩]**
*   **关系**: **雇佣兵 -> 死忠**。顾子轩救了他一命，并帮他解除了泰坦的控制项圈。
*   **互动模式**: 沉默的守护。只要顾子轩一个眼神，陆霆就会扫平前方的障碍。

### 敌对阵营张力 (Antagonist Tensions)

**[马木] -> [林野]**
*   **关系**: **傲慢的俯视 -> 意外的变量**。最初根本看不起这个修电器的，认为他只是时代的灰烬。
*   **冲突点**: 马木试图用“完美的秩序”拯救世界，而林野代表着“混乱的生命力”。

**[李维] -> [马木]**
*   **关系**: **狂热信徒**。李维是马木意志的绝对执行者，但他比马木更冷血，有时甚至觉得马木还不够“绝对”。

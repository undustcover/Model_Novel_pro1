# 人物档案：王苟 (Wang Gou)

## 1. 基础信息 (Basic Info)
*   **姓名**: 王苟 (Wang Gou)
*   **外号**: 胖子、苟王、人肉盾牌（自嘲）
*   **年龄**: 17岁
*   **性别**: 男
*   **职业**: 联邦第三理工附中 高二(3)班学生 / 林野的死党
*   **阵营**: 守序中立 -> 混乱善良

## 2. 外貌特征 (Appearance)
*   **体型**: 175cm，体重100kg。圆滚滚的体型，皮肤白嫩，看起来人畜无害。
*   **觉醒后特征**: 发动异能时，体型会瞬间膨胀至3米，皮肤表面覆盖一层黑色的生物角质层，像是一只直立行走的装甲暴龙。
*   **衣着风格**: 总是穿着最大号的校服，因为经常吃零食，衣服上总有不明油渍。

## 3. 性格内核 (Personality Core)
*   **MBTI**: ESFP / 享乐主义者
*   **核心关键词**: 胆小、讲义气、吃货
*   **优点**: **绝对忠诚**。平时看起来胆小怕事，一有风吹草动就想跑，但只要林野或者苏浅遇到危险，他会毫不犹豫地挡在前面。
*   **弱点**: **饥饿感**。异能消耗极大，需要不停地进食。如果过度饥饿，会陷入失控的暴食状态，敌我不分。
*   **口头禅**: “完了完了，这次死定了……诶？野哥你有办法？那没事了。”

## 4. 背景故事 (Backstory)
*   **平庸的胖子**: 班级里的透明人B，因为肥胖常年被欺负，只有林野把他当朋友，帮他修好了被踩坏的游戏机。从此死心塌地跟着林野。
*   **觉醒契机**: 在第 2 章的课堂上，因接触到“深海论坛”的高维模因，差点变异。被苏浅用钢笔砸醒（物理打断施法），但也因此因祸得福，体内残留了部分高维能量，觉醒了肉体强化类异能。

## 5. 异能/战斗设定 (Abilities)
*   **核心异能**: **暴食装甲 (Gluttony Armor)** - 本质：生物质能转化与硬化
    *   *原理*: 胃部变异为一个微型“生物反应堆”，能瞬间将摄入的有机物转化为高密度的生物护盾。
*   **技能树**:
    *   **肉弹战车 (Lv.1)**: 将身体蜷缩成球，利用体重和动能进行冲撞。
    *   **硬化皮肤 (Lv.1)**: 短时间内让皮肤硬度达到防弹钢板级别，能硬抗手枪子弹。
    *   **吞噬恢复 (Lv.2)**: (中期解锁) 通过吞噬敌人的生物组织或泰坦的生物电池，快速恢复伤势。

## 6. 在主角团的定位
*   **坦克 (Tank)**: 负责吸收伤害，保护后排的林野（控制/输出）和艾薇（黑客）。
*   **吉祥物/气氛组**: 在队伍气氛沉重时，负责插科打诨，缓解压力。

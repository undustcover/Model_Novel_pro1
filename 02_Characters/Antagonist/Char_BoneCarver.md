# 人物档案：剔骨刀 (Bone Carver)

## 1. 基础信息 (Basic Info)
*   **真实姓名**: 已抹除 (代号：Unit-7)
*   **外号**: 剔骨刀
*   **身份**: 泰坦安保部 特别行动组组长 / 改造人
*   **年龄**: 不详
*   **阵营**: 守序邪恶 (只听命于李维和马木)

## 2. 外貌与特征
*   **外貌**: 全身90%以上义体化。没有皮肤，只有冰冷的碳纤维外壳。面部是一个红色的电子眼，像是一个单眼恶魔。
*   **武器**: 双臂内置高频振动刀，能切开坦克的装甲。背部挂载微型导弹巢。
*   **风格**: 沉默寡言，从不废话。任务失败会毫不犹豫地自爆。

## 3. 战斗风格
*   **速度型刺客**: 拥有光学迷彩，能隐形接近目标。
*   **杀戮机器**: 战斗时没有任何感情波动，计算出最优的杀戮路径。

## 4. 剧情作用
*   **物理威胁**: 前期追杀主角团的主要战力。
*   **压迫感来源**: 每一次出场都代表着死亡和破坏。

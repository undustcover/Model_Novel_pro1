
## 系统全景概览
#### 1. 🧠 大脑：系统指令 ( 00_System_Prompts )
- Core_Instruction.md : 包含 三条铁律 （逻辑优先、定稿优先、角色一致性）。AI 每次工作前都会“阅读”此文件，确保它知道自己是一个严肃的合著者，而不是简单的文本生成器。
- Style_Guide.md : 定义了“冷峻科幻 + 烟火都市”的混合文风，强调 感官描写 和 潜台词对话 。
#### 2. 🌏 基石：世界观圣经 ( 01_World_Bible )
- World_Rules_Physics.md (硬限制) : 定义了异能的 代价 （如脱水、精神侵蚀）。AI 会明白“力量从不免费”。
- World_Rules_Society.md (软限制) : 记录了杀戮、暴露身份的 社会性后果 。确保故事具有现实沉重感。 
#### 3. 👥 灵魂：人物与关系 ( 02_Characters )
- Relationship_Graph.md : 这是本系统的亮点。它不只记录谁认识谁，还记录了**“利益共同点” 和 “深层矛盾”**，专为群像戏设计。
- Template_Char.md : 包含 MBTI、核心欲望、恐惧之物等深度心理字段。
#### 4. 📝 骨架：剧情规划 ( 03_Story_Planning )
- Plot_Threads_Tracker.md : 伏笔回收表 。AI 会定期检查这里，看看有没有机会填坑。
- Story_State_Snapshot.md : 剧情快照 。记录每一卷结束时的世界状态（谁受了伤、谁拿着关键道具），防止长篇连载出现记忆偏差。
#### 5. ✍️ 手部：写作工作区 ( 04_Writing_Workspace )
- Reference_Samples/ : 请把您觉得写得好的片段丢进去，AI 会模仿这些“高分样本”。
- Volume_01_Drafts/ : 采用双文件流—— Draft_AI.md (AI 草稿) 和 Final_User.md (用户定稿)。

##  🚀 快速启动指南
建议您按照以下顺序开始创作：

#### 1. 填充世界观 
打开 01_World_Bible 中的两个文档，把您脑海中的“异能设定”和“社会规则”填进去。
#### 2. 建立人物 
在 02_Characters 中复制模板，创建 1 个主角和 1 个主要反派的档案。
#### 3. 理清关系 
在 Relationship_Graph.md 中写下他们最初的矛盾。
#### 4. 投喂样本 
在 04_Writing_Workspace/Reference_Samples 里放入一段您喜欢的科幻/都市小说片段。
#### 5.开始大纲 
在 03_Story_Planning 里写下第一卷的粗略想法，然后对 AI 说：
   “请读取世界观和人物卡，帮我完善第一卷的分章细纲，重点体现主角在第 3 章的心理转变。”

## 系统规则
#### 方式一：自动生效（日常使用）
通常情况下，您 不需要做任何事 。只要这个文件在根目录，AI 就会变聪明。

- 测试方法 ：您可以直接问 AI：“我想写一场主角在雨巷里的战斗，请帮我生成草稿。”
- 预期反应 ：AI 会自动根据 .cursorrules 的指引，去读取您的 Style_Guide.md 和 Protagonist 档案，然后生成符合“感官描写要求”的文本。 
#### 方式二：手动唤醒（双重保险）
如果某些时候 AI 似乎“忘记”了设定（比如上下文过长导致遗忘），您可以输入以下简短的**“唤醒咒语”**：

“加载系统设定。”

或者：

“@.cursorrules 初始化当前会话。”

这会强制 AI 重新扫描一遍规则文件。

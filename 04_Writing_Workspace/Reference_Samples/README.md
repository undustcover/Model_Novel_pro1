# 风格样本库 (Reference Samples Library)

> **用途**: 在此文件夹中放入您认为写得极好的小说片段（Markdown格式）。AI 将学习这些片段的用词、节奏和描写手法。

## 使用方法
1.  创建一个 `.md` 文件，例如 `Cyberpunk_Combat_Sample.md`。
2.  将您喜欢的段落粘贴进去。
3.  在 Prompt 中告诉 AI：“请参考 `Cyberpunk_Combat_Sample.md` 的风格来写这一场战斗。”

## 推荐分类
建议建立以下几类样本：
*   **Combat.md**: 战斗描写样本（侧重动作感、打击感）。
*   **Dialogue.md**: 对话描写样本（侧重潜台词、群像互动）。
*   **Environment.md**: 环境描写样本（侧重氛围渲染）。
*   **Emotion.md**: 心理描写样本（侧重细腻情感）。

# 人物档案：王苟 (Wang Gou)

## 1. 基础信息 (Basic Info)
*   **姓名**: 王苟 (Wang Gou)
*   **外号**: 胖子、苟王、人肉盾牌（自嘲）
*   **年龄**: 17岁
*   **性别**: 男
*   **职业**: 联邦第三理工附中 高二(3)班学生 / 林野的死党
*   **身份**: 
    *   **表**: 依靠泰坦奖学金生存的C级公民，凯恩的“模范宠物”。
    *   **里**: 已故工程师王大锤之子，莫瞎子的关门弟子，地下管网的活地图。
*   **阵营**: 守序中立 (伪装) -> 混乱善良 (觉醒)

## 2. 外貌特征 (Appearance)
*   **体型**: 175cm，体重100kg。圆滚滚的体型，皮肤白嫩，看起来人畜无害。
*   **觉醒后特征**: 发动异能时，体型会瞬间膨胀至3米，皮肤表面覆盖一层黑色的生物角质层，像是一只直立行走的装甲暴龙。
*   **衣着风格**: 总是穿着最大号的校服，因为经常吃零食，衣服上总有不明油渍。
*   **随身物品**: 
    *   一枚擦得锃亮的**泰坦优秀市民勋章**（挂在胸前，保命符）。
    *   一把老旧的**机械游标卡尺**（藏在贴身口袋，父亲遗物）。

## 3. 性格内核 (Personality Core)
*   **MBTI**: ESFP / 享乐主义者 (表象) -> ISFP / 探险家 (内在)
*   **核心关键词**: **生存、伪装、守护**
*   **优点**: **绝对忠诚**。平时看起来胆小怕事，一有风吹草动就想跑，但只要林野或者苏浅遇到危险，他会毫不犹豫地挡在前面。
*   **弱点**: **饥饿感**。异能消耗极大，需要不停地进食。
*   **口头禅**: “嘿嘿，凯恩长官说得对，活着比什么都重要。”（对泰坦人员）；“完了完了，这次死定了……野哥救我！”（对林野）。

## 4. 背景故事 (Backstory) - [重构版]
*   **童年阴影 (The Fall)**: 
    *   8岁前，父亲王大锤是他的英雄。
    *   8岁那年，目睹父亲为了“赎罪”跪在凯恩面前，并被凯恩处决。
    *   凯恩递给他一块面包，告诉他：“想活下去，就学会跪着。” 他接过了面包。
*   **金丝笼 (The Cage)**:
    *   在凯恩的“资助”下长大。凯恩不断羞辱他，打压他的自尊，试图证明“英雄的儿子也只是狗”。
    *   **驯化事件**:
        *   **枪的测试**: 13岁时，凯恩故意给他一把枪让他杀欺负他的同学。王苟通过“尿裤子”的表演放弃了复仇，骗过了凯恩。
        *   **公开感谢**: 在泰坦庆典上，声泪俱下地感谢杀父仇人凯恩，甚至在心里咬破舌尖来维持那种“感激涕零”的假象。
*   **地下导师 (The Mentor)**:
    *   在第9区边缘拾荒时，遇到了盲人拾荒者**莫瞎子**（前反抗军成员）。
    *   莫瞎子看穿了他的伪装，教他在下水道生存，教他识字读史。
    *   15岁时，莫瞎子为了保护他而被迫离开第9区。离别前留给他一张地下管网图和一张与父亲王大锤的合影。这成为了王苟彻底蜕变的转折点——他戴上了“小丑”的面具，在凯恩面前演戏，暗中积蓄力量。
*   **觉醒契机**: 
    *   课堂上的高维模因事件是导火索，但他体内沉睡的“失落帝国血脉”（遗传自父亲）才是根本。

## 5. 异能/战斗设定 (Abilities)
*   **核心异能**: **暴食装甲 (Gluttony Armor)** - 本质：生物质能转化与硬化
*   **技能树**:
    *   **肉弹战车 (Lv.1)**: 将身体蜷缩成球，利用体重和动能进行冲撞。
    *   **硬化皮肤 (Lv.1)**: 短时间内让皮肤硬度达到防弹钢板级别。
    *   **吞噬恢复 (Lv.2)**: (中期解锁) 通过吞噬敌人的生物组织或泰坦的生物电池，快速恢复伤势。

## 6. 在主角团的定位
*   **坦克 (Tank)**: 负责吸收伤害。
*   **活地图 (Navigator)**: 依靠父亲留下的《旧城管网手绘图》和莫瞎子的教导，能带领团队在地下迷宫中穿梭。
*   **人性锚点**: 用最卑微的姿态，守护最硬的道理。

# 人物档案：王大锤 (Wang DaChui)

## 1. 基础信息 (Basic Info)
*   **姓名**: 王大锤 (Wang DaChui)
*   **身份**: 前泰坦重工高级工程师 / 拥有“失落人类帝国一级权限”的觉醒者
*   **状态**: 已故 (9年前)
*   **关系**: 
    *   **王苟**: 儿子，希望能像普通人一样活着。
    *   **凯恩**: 挚友、执行者。

## 2. 性格与信念
*   **理想主义者**: 相信科技应该服务于人类，而不是奴役人类。
*   **矛盾体**: 一方面为泰坦工作，建设城市；另一方面对泰坦清洗异能者的暴行感到恐惧和愤怒。
*   **父爱如山**: 为了儿子，甘愿背负“叛徒”或“懦夫”的骂名。

## 3. 关键经历
*   **觉醒**: 在工作中意外发现自己对古老机械有着超乎常理的亲和力（机械专精异能）。随后通过查阅绝密档案，得知这是“失落人类帝国”的权限赋予。
*   **真相**: 发现泰坦所谓“清除异能者”的本质是在清除“旧人类文明的继承者”。他曾是**反抗军第9小组**的核心成员，代号“锤子”，与莫瞎子是战友。
*   **托孤**: 察觉到自己已经被泰坦的监控算法锁定（或者王苟也表现出了异常）。为了打破必死的局面，他找到了凯恩，制定了“牺牲计划”。
    *   *计划内容*: 由凯恩亲手处决“失控”的他，以此换取凯恩的晋升和对王苟的“合法监护权”。
*   **遗产**: 
    *   **《旧城管网手绘图》**: 记录了泰坦监控盲区和逃生通道。
    *   **机械卡尺/笔记本**: 藏有关于“一级权限”的加密信息（王苟目前无法解读，仅作为念想）。

## 4. 剧情作用
*   **背景板**: 他的死是第11-13章的核心驱动力。
*   **世界观揭秘**: 通过他的笔记（后续剧情），逐步揭开异能与失落帝国的关系。
